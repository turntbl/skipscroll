## ⚠️ Common SEO Pitfalls to Avoid

1. **Don't Over-Optimize**
   - Keep keyword density natural (1-2%)
   - Don't stuff keywords in headings
   - Write for humans first, search engines second

2. **Don't Neglect Mobile**
   - Test on real devices
   - Optimize images for mobile
   - Ensure touch targets are large enough

3. **Don't Ignore Page Speed**
   - Optimize images (use WebP)
   - Minimize JavaScript
   - Use CDN for assets

4. **Don't Buy Links**
   - Focus on earned, organic backlinks
   - Quality over quantity
   - Avoid link schemes

5. **Don't Forget User Intent**
   - Match content to search intent
   - Provide actionable value
   - Answer questions completely
